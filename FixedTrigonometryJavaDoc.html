Class FixedTrigonometry

Package: default

Author: Mario Gianota

Overview

The FixedTrigonometry class provides a standard suite of trigonometric functions computed entirely through FixedArithmetic operations. It avoids all native Math.* calls, floating-point arithmetic, and hardware-specific trig instructions to ensure bit-for-bit determinism across all platforms.

The implementation uses Taylor series expansions for sine and cosine, combined with quadrant-based range reduction to maintain high precision and fast convergence.

Example Usage

// Define an angle in radians (e.g., PI / 4)
FixedArithmetic angle = FixedTrigonometry.PI.divide(FixedArithmetic.of(4));

// Compute sine and cosine
FixedArithmetic sineVal = FixedTrigonometry.sin(angle);
FixedArithmetic cosVal  = FixedTrigonometry.cos(angle);

// Compute inverse tangent (atan2) for coordinates (1, 1)
FixedArithmetic y = FixedArithmetic.of(1);
FixedArithmetic x = FixedArithmetic.of(1);
FixedArithmetic theta = FixedTrigonometry.atan2(y, x);

System.out.println("Sin(PI/4): " + sineVal);
System.out.println("Atan2(1,1): " + theta);

Field Summary

Method Summary

public static FixedArithmetic sin(FixedArithmetic x)

Computes the sine of an angle x (in radians). Uses range reduction to map the input to [-π/4, π/4] before applying the Taylor series expansion.

public static FixedArithmetic cos(FixedArithmetic x)

Computes the cosine of an angle x (in radians). Internally leverages the sine implementation with a phase shift (sin(x + π/2)) or identities for efficiency.

public static FixedArithmetic tan(FixedArithmetic x)

Returns the tangent of an angle x. Computed as sin(x) / cos(x) using deterministic fixed-point division.

public static FixedArithmetic asin(FixedArithmetic x)

Computes the arcsine (inverse sine) of x. The result is in the range [-π/2, π/2] radians.

public static FixedArithmetic acos(FixedArithmetic x)

Computes the arccosine (inverse cosine) of x. The result is in the range [0, π] radians.

public static FixedArithmetic atan(FixedArithmetic x)

Computes the arctangent (inverse tangent) of x. The result is in the range (-π/2, π/2) radians.

public static FixedArithmetic atan2(FixedArithmetic y, FixedArithmetic x)

Computes the two-argument arctangent of y and x. This method handles quadrant detection and prevents division-by-zero errors when calculating the angle of a vector.
