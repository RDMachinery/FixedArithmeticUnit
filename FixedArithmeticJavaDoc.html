Class FixedArithmetic

Package: default

Author: Mario Gianota

Overview

FixedArithmetic is a specialized utility class designed for high-precision, deterministic arithmetic using fixed-point representation. It is engineered specifically for environments like scientific computing and quantitative trading where floating-point non-determinism (IEEE-754) is unacceptable.

The arithmetic core is unique in that it performs addition, subtraction, multiplication, and division without using the CPU's native multiplication, division, or modulo operators. Instead, it relies solely on repeated integer addition and subtraction to ensure identical results across different hardware architectures.

Example Usage

// Creating values from strings or integers
FixedArithmetic price = FixedArithmetic.of("100.50");
FixedArithmetic quantity = FixedArithmetic.of(5);

// Chained operations: (7 + 3) * (10 - 4) / 5
FixedArithmetic result = FixedArithmetic.of(7)
    .add(FixedArithmetic.of(3))
    .multiply(FixedArithmetic.of(10).subtract(FixedArithmetic.of(4)))
    .divide(FixedArithmetic.of(5));

System.out.println("Result: " + result.toString()); // Output: 12.0

Method Summary

public static FixedArithmetic of(long integerValue)

Factory method to construct a new FixedArithmetic instance from a plain long integer.

Parameters: integerValue - The whole number to represent.

public static FixedArithmetic of(String decimal)

Constructs an instance from a decimal string (e.g., "3.14"). This method parses the string character-by-character using only integer addition and subtraction, ensuring no floating-point "poisoning" occurs during construction.

Throws: IllegalArgumentException if the string format is invalid or has multiple decimal points.

public FixedArithmetic add(FixedArithmetic other)

Returns a new instance representing the sum (this + other).

public FixedArithmetic subtract(FixedArithmetic other)

Returns a new instance representing the difference (this - other).

public FixedArithmetic multiply(FixedArithmetic other)

Returns a new instance representing the product (this * other). It utilizes a Russian Peasant (binary) multiplication algorithm built entirely from addition to maintain hardware-independent determinism.

public FixedArithmetic divide(FixedArithmetic other)

Returns a new instance representing the quotient (this / other). It employs a long-division-by-repeated-subtraction algorithm, processing the quotient digit-by-digit to ensure precision.

Throws: ArithmeticException on division by zero.

public long integerPart()

Returns the whole number part of the stored value, truncated toward zero.

public long remainder()

Returns the fractional remainder in units of 10^-PRECISION.

public String toString()

Returns a human-readable fixed-point string representation of the value, with trailing zeros trimmed for clarity.
